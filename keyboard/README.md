# Shortcuts

MacOS:
- Hide window

Magnet:
- Maximize: ⌃ ⌥ + ⏎
- Left: ⌃ ⌥ + ←
- Right: ⌃ ⌥ + →
- Left third:
- Right third
- Left two thirds
- Right two thirds

Flycut:
- Copy: ⌘ ⇧ + c
- Paste: ⌘ ⇧ + v

Open Apps:
- Safari: ⇪ + s
- Discord: ⇪ + d
- VS Code: ⇪ + w
- Terminal: ⇪ + e
- Tower: ⇪ + r
- Chrome: ⇪ + f
- Studio 3T: ⇪ + q

Other:
- Dictation Key (F5): Toggle Discord mute
- F3: backlight down
- F4: backlight up
- Next/Prev tab across apps
